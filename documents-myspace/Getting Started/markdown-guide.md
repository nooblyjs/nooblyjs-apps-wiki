# Markdown Quick Reference

A quick guide to markdown formatting in your wiki.

## Headers

```
# H1 Header
## H2 Header
### H3 Header
```

## Text Formatting

- **Bold text**: `**bold**`
- *Italic text*: `*italic*`
- ~~Strikethrough~~: `~~strikethrough~~`
- `Code`: `` `code` ``

## Lists

### Unordered Lists
```
- Item 1
- Item 2
  - Nested item
```

### Ordered Lists
```
1. First item
2. Second item
3. Third item
```

## Links and Images

- Link: `[Link Text](https://example.com)`
- Image: `![Alt Text](image-url.jpg)`

## Code Blocks

\`\`\`javascript
const example = "code block";
\`\`\`

## Tables

```
| Column 1 | Column 2 |
|----------|----------|
| Data 1   | Data 2   |
```

## Blockquotes

```
> This is a quote
```
